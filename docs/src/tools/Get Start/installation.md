# Installation

## 📦 安装

### @fireflymit/ui

::: code-group

```bash [pnpm]
pnpm add @fireflymit/ui
```

```bash [yarn]
yarn add @fireflymit/ui
```

```bash [npm]
npm install @fireflymit/ui
```

:::
